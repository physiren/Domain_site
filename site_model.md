# CIPS / CIPSe 格点模型说明：从 DFT 前 toy 测试到代码实现

## 0. 文档目标

本文单独讨论一条与连续 LGD 模型不同、但互补的建模路线：

* 以 **Cu 离子局域位移** 为核心自由度；
* 在二维格点上构建 **局域势 + 短程相互作用 + 长程偶极相互作用** 的有效哈密顿量；
* 在正式进行 DFT 参数提取之前，先用一套 **toy 参数** 做数值试验，检验模型是否能够自然产生我们关心的现象；
* 最后给出一个面向 code agent 的实现章节，使其可以直接开始写代码。

本文的目标不是给出唯一正确的显微模型，而是提供一套：

1. **物理上合理**；
2. **参数意义清楚**；
3. **便于后续用 DFT 结果逐步替换 toy 参数**；
4. **能够尽快在计算机中验证机制可行性**

的格点模型方案。

---

## 1. 为什么考虑格点模型

连续模型擅长描述：

* 有限波矢调制倾向；
* 功率谱与结构因子；
* 外场与温度下的宏观畴统计性质。

但对本体系而言，还存在一类很自然的显微图像：

* Cu 离子在局域多阱势中发生位移；
* 相邻格点之间通过晶格畸变、局域化学与偶极相互作用耦合；
* 不同材料（CIPS 与 CIPSe）之间的差异，本质上体现在局域势深浅、近邻协同、竞争相互作用以及偶极强度上。

因此，将每个格点上的序参量取为一个局域变量，例如

[
s_i \sim z_i,
]

是很自然的。这里：

* (i) 标记二维格点；
* (s_i) 可以理解为 Cu 局域面外位移的无量纲化形式；
* (s_i>0) 与 (s_i<0) 分别代表两个相反的局域极化方向；
* (|s_i|\sim 1) 代表“极化谷底附近”的典型幅值。

这条路线的优势是：

* 直接对应显微图像；
* 便于从 DFT 提取 onsite 势和有限个短程耦合；
* 便于后续做 Monte Carlo、动力学演化与畴粗化模拟；
* 对 FE/AFE 竞争、条纹态、迷宫态、畴壁、滞回都比较自然。

## 2. 最小格点模型

### 2.1 单自由度二维软自旋模型

建议首先使用如下最小哈密顿量：

[
H=
\sum_i V_{\rm on}(s_i)
-J_1\sum_{\langle ij\rangle}s_is_j
+J_2\sum_{\langle\langle ij\rangle\rangle}s_is_j
+\frac12\sum_{i\neq j} D_{\rm eff},\frac{s_is_j}{r_{ij}^3}
-h\sum_i s_i.
]

这里各项的含义是：

* (V_{\rm on}(s_i))：局域多阱势；
* (J_1)：最近邻短程耦合，取正时偏向同向有序；
* (J_2)：次近邻竞争项，取正时可与 FE 型 (J_1) 竞争；
* (D_{\rm eff})：长程偶极相互作用有效强度；
* (h)：外场耦合项。

### 2.2 局域势的最小形式

建议先用六次多项式：

[
V_{\rm on}(s)=\frac{\alpha(T)}{2}s^2-\frac{\beta}{4}s^4+\frac{\gamma}{6}s^6.
]

其中：

* (\alpha(T)) 控制顺电到局域极化的软化；
* (\beta>0) 与 (\gamma>0) 一起形成双阱或弱一阶多阱势；
* 通常可先取 (\gamma=1) 作为能量单位归一化的一部分。

### 2.3 符号约定

本文统一采用：

* (-J_1 s_i s_j)：因此 (J_1>0) 偏向 FE 同向排列；
* (+J_2 s_i s_j)：因此 (J_2>0) 可作为与 (J_1) 竞争的项；
* 偶极项写成最简形式的有效各向同性 (1/r^3) 核，仅作为 toy 阶段使用。

正式建模时，若需要更真实，可将偶极核改成带角度因子的张量形式。

## 2A. 基于当前 POSCAR 的几何修正：单层应按三角晶格处理

根据当前 POSCAR，面内两个基矢等长，夹角 120 度，因此单层的面内 Bravais 晶格应视为三角晶格，而不是方格子。当前 primitive cell 中有两层 Cu，分别位于两个互补的面内位置；如果第一阶段先做单层 toy 模型，则最自然的抽象是：每一层各自对应一个单原子基底的三角晶格，单层 primitive cell 中只保留一个 Cu 自由度。

这会直接改变 toy 模型中“高对称图样”的选择。

### 2A.1 为什么不能再用 checkerboard

在方格子上，最近邻反平行可以形成简单 checkerboard；但在三角晶格上，如果序参量仍是单标量，那么“所有最近邻都反平行”在几何上是挫折的，不能由一个简单二分子格实现。因此，先前按方格子写的 checkerboard、stripe_x、stripe_y 不再是最合适的主参考图样。

### 2A.2 三角晶格上更合理的最小高对称图样

对当前单标量软自旋模型，建议至少比较以下三类图样：

1. Gamma 点均匀态：所有格点同号，对应均匀 FE 或 FiE 倾向。
2. M 点条纹态：沿某一晶向，同一条链上同号，相邻链之间反号。这是三角晶格上最常见、最低成本的非均匀参考态。
3. K 点三子晶格态：需要 sqrt(3)×sqrt(3) 的三子晶格超胞。对单标量 toy 模型，可以先用 (A, A, -A) 或 (A, 0, -A) 这一类标量三子晶格构型作为最小代理，而不必一开始就追求矢量 120 度自旋序。

因此，对当前体系更合适的最小超胞与扫描图样是：

* 1×1 primitive：用于 onsite 和均匀 FE；
* 2×1 或 2×2 supercell：用于 M 点条纹态；
* sqrt(3)×sqrt(3)R30° supercell：用于 K 点三子晶格态。

如果第一阶段目标只是区分 CIPS 与 CIPSe 是否更容易出现有限尺度竞争纹理，那么先比较 Gamma 与 M 已经很有用；如果 K 点构型也显著降能，则说明系统可能不仅仅是简单条纹倾向，而是更强的三子晶格竞争。

### 2A.3 对参数解释的修正

在三角晶格上，参数含义本身不变：

* V_on：局域 Cu 多阱势；
* J1：最近邻协同项；
* J2：次近邻竞争项；
* D_eff：长程偶极；
* h：外场。

但现在要特别强调：

* 不能再把“最近邻反平行”简单表述成 checkerboard；
* 若 toy 模型中出现稳定有限尺度纹理，其最可能对应的是 M 点条纹、K 点三子晶格调制，或二者与长程偶极共同形成的迷宫或条纹混合态。

## 3. 这个模型能描述哪些物理

在适当参数区间下，这个模型可以描述：

1. **顺电到局域极化有序转变**；
2. **均匀 FE / FiE 态**；
3. **AFE / 交错有序态**；
4. **条纹、迷宫、有限尺度纹理**；
5. **畴壁与畴粗化**；
6. **在外场扫描下的滞回**；
7. **降温增强并在低温饱和的非均匀散射通道**。

尤其对我们当前问题，它的价值在于：

* 可以直接比较 CIPS 与 CIPSe 是否具有不同的“形成复杂畴纹理的倾向”；
* 可以非常快速地测试：**只靠局域势差异是否足够，还是必须引入竞争项 (J_2) 或偶极项 (D_{\rm eff})**；
* 可以在 DFT 参数尚未齐备前，用 toy 参数先验证“机制能否跑通”。

## 4. 与材料物理的对应关系

### 4.1 (V_{\rm on})：Cu 的局域多阱势

这项代表：

* Cu 在局域环境中的位移势阱；
* 正负极化构型之间的局域势垒；
* 局域极化是否“深、刚、单一”还是“浅、软、多亚稳态”。

对 CIPS 与 CIPSe 的最直观区分，可以首先体现在这一项上：

* **CIPS**：可先取更深的局域势阱；
* **CIPSe**：可先取更浅、更易受竞争项重整化的局域势阱。

### 4.2 (J_1)：最近邻协同晶格畸变

这项最自然地理解为：

* 局域化学与共享原子导致的近程协同位移；
* 晶格畸变与局域弹性能对相邻 Cu 位移一致性的偏好。

它通常是“让体系更像普通 FE / FiE”的主力量之一。

### 4.3 (J_2)：中程竞争

这项代表：

* FE 与 AFE 排列之间的中程竞争；
* 由更远一点的短程晶格相互作用引起的挫折；
* 促成条纹、交错或有限尺度纹理的最简单机制。

从建模角度，它是最便宜、最有效的“复杂纹理种子”。

### 4.4 (D_{\rm eff})：长程偶极相互作用

这项代表：

* 由局域极化产生的长程电偶极相互作用；
* 对真实畴形成与尺度选择非常关键；
* 与层状薄膜/二维材料中的边界条件耦合密切相关。

在 toy 阶段，我们先用最简各向同性 (1/r^3) 核。后续若要更真实，可以替换成带角度依赖、介电屏蔽和厚度截断的核。

### 4.5 (h)：外场

这项直接表示实验中的栅压或等效场偏置。它的作用是：

* 推动均匀极化；
* 抑制复杂畴纹理；
* 产生扫场滞回与畴结构重排。

## 5. Toy 参数为什么要先做

在正式 DFT 之前先做 toy 参数扫描，有三个目的。

### 5.1 验证机制可行性

先回答：

* 只靠局域势差异，是否就能让 CIPS 与 CIPSe 出现明显不同；
* 是否必须加入 (J_2) 才会有复杂纹理；
* 偶极项对纹理尺度是否关键；
* 哪个参数最敏感。

### 5.2 降低后续 DFT 压力

若 toy 扫描表明某些参数根本不敏感，后续 DFT 就不必为这些参数花很多算力。

### 5.3 给代码开发一个稳定目标

先用 toy 参数把代码流程、观测量、可视化和分析管线都跑通。这样 DFT 结果出来后只需要“换参数”，不用重写模型。

## 6. Toy 参数的推荐量级与物理意义

下面给出一套适合第一阶段数值试验的参数区间。约定所有能量都用 **meV / site**。

### 6.1 局域势深度

将双阱深度近似记作 (\Delta_{\rm loc})。建议先取：

* **CIPS**：(\Delta_{\rm loc}\sim 80)–(250) meV/site
* **CIPSe**：(\Delta_{\rm loc}\sim 40)–(180) meV/site

对应的物理意义是：

* CIPS 先视作更深、更刚的局域极化势阱；
* CIPSe 先视作更浅、更软，更容易受竞争项影响。

若使用

[
V_{\rm on}(s)=\frac{\alpha}{2}s^2-\frac{\beta}{4}s^4+\frac{\gamma}{6}s^6,
]

可先固定 (\gamma=1)，并尝试：

* **CIPS**：(\alpha=1.0)–(2.0)，(\beta=2.5)–(4.0)
* **CIPSe**：(\alpha=0.6)–(1.5)，(\beta=2.0)–(3.2)

这些值的目的不是拟合真实材料，而是保证双阱存在且两种材料呈现“深 vs 浅”的对照。

### 6.2 最近邻耦合 (J_1)

建议先扫：

* **CIPS**：(J_1=8)–(20) meV
* **CIPSe**：(J_1=5)–(15) meV

物理意义：

* CIPS 更偏向稳定均匀有序；
* CIPSe 的局域有序协同稍弱，更容易被竞争项打断。

### 6.3 次近邻竞争项 (J_2)

建议重点扫比值：

* **CIPS**：(J_2/J_1=0)–(0.15)
* **CIPSe**：(J_2/J_1=0.15)–(0.5)

物理意义：

* CIPS 先放在“弱竞争”一侧；
* CIPSe 先放在“中等到较强竞争”一侧。

这是 toy 阶段最重要的区分之一，因为很多复杂纹理的产生首先依赖“足够的竞争”。

### 6.4 长程偶极强度 (D_{\rm eff})

建议先取：

* **CIPS**：(D_{\rm eff}=5)–(30) meV
* **CIPSe**：(D_{\rm eff}=2)–(15) meV

物理意义：

* CIPS 通常极化更强，因此偶极作用也可先取更大；
* CIPSe 偶极项略小，但更强竞争可能弥补甚至压过这个差异。

### 6.5 外场 (h)

先扫：

[
h=0\text{–}20\ \text{meV/site}
]

物理意义：

* 小 (h)：接近弱场、复杂纹理易出现；
* 大 (h)：推动均匀 FE，抑制复杂纹理。

J_1=8\ \text{meV},\quad

## 8. Toy 测试时优先观察哪些量

第一阶段不要先盯着“是否完美拟合实验曲线”，而应优先看下列观测量。

### 8.1 平均极化

[
\bar s=\frac{1}{N}\sum_i s_i
]

意义：判断体系整体偏向均匀 FE 还是仍保留大量正负相抵消的纹理。

### 8.2 结构因子 / 功率谱

[
S(\mathbf q)=\left|\sum_i (s_i-\bar s)e^{-i\mathbf q\cdot \mathbf r_i}\right|^2
]

意义：

* 若权重集中在 (q\approx 0)，说明更像均匀 FE / 大畴；
* 若在某个有限 (q) 环或带上有显著权重，说明存在有限尺度纹理。

### 8.3 径向平均谱

[
S(q)=\langle S(\mathbf q)\rangle_{|\mathbf q|=q}
]

意义：最直接对应我们关心的“功率谱增强、是否出现有限尺度、是否低温饱和”。

### 8.4 畴尺度

可从谱峰位置定义近似畴尺度：

[
L_{\rm dom}\sim \frac{2\pi}{q_{\rm peak}}.
]

### 8.5 滞回

在扫场协议下记录：

* (\bar s(h))
* 结构因子峰强随 (h) 的变化

意义：判断复杂纹理是否在强场下被压回均匀态。

## 9. Toy 阶段的判据：什么算成功

若数值模拟能表现出如下对照，则说明模型路线值得继续：

### 9.1 CIPS

* 更容易进入均匀 FE 或大畴状态；
* 功率谱主要在低 (q) 附近增强；
* 强场下快速变成几乎无非均匀散射的态。

### 9.2 CIPSe

* 更容易出现条纹/迷宫/有限尺度纹理；
* 功率谱在某个有限 (q) 区间有明显权重；
* 降温时非均匀散射增强，低温后幅值饱和；
* 强场下这些纹理被压制。

如果出现上述对照，即使参数还只是 toy 级别，也已经说明：

* 格点模型本身能承载我们关心的机制；
* 后续用 DFT 替换参数是值得的。

## 10. 与 DFT 的衔接方式

本模型中各类参数未来可按如下顺序替换。

### 10.1 onsite 势 (V_{\rm on})

由 DFT 的 frozen-phonon 或沿 Cu 主位移路径的受限总能得到。

### 10.2 (J_1)、(J_2)

由 2×2 或更大超胞下的高对称图样能量差提取。

### 10.3 (D_{\rm eff})

可由：

* 自发极化；
* Born 有效电荷；
* 介电常数；
* 晶格常数；

估算并固定到合理区间，再用 toy/拟合做小范围修正。

### 10.4 外场映射

将实验门电压或等效场转换为模型中的 (h)。初期不求精确，只需要能保证“弱场有纹理、强场趋于均匀态”的层级关系。

### 11.5 第一版推荐算法

建议先用 **Metropolis Monte Carlo**：

* 简单；
* 能直接扫温度；
* 能观察相变、结构因子、滞回；
* 对软自旋变量可用局域随机提案。

#### 单步更新示意

对每个格点 ((i,j))：

1. 提议
   [
   s_{ij}' = s_{ij} + \delta,
   ]
   其中 (\delta\sim \mathrm{Uniform}[-\Delta,\Delta])

2. 计算能量差 (\Delta H)

3. 以概率
   [
   p=\min\left(1,e^{-\Delta H/(k_BT)}\right)
   ]
   接受

### 11.6 能量计算

建议把能量拆成函数：

* `onsite_energy(s, alpha, beta, gamma)`
* `nn_energy(s, J1, neighbors_1)`
* `nnn_energy(s, J2, neighbors_2)`
* `dipolar_energy(s, D_eff, positions)`
* `field_energy(s, h)`

对长程偶极项，第一版可先：

* 用实空间求和 + 截断半径；
* 或用 FFT 卷积加速。

若体系不太大（例如 (64\times64) 或 (128\times128)），实空间截断求和就足够做 toy 测试。

### 11.7 温度协议

建议先实现：

1. 高温随机初态；
2. 逐步降温；
3. 每个温度点充分热化后采样；
4. 将上一温度终态作为下一温度初态。

这可以最直接地对应实验中的冷却过程。

### 11.8 外场协议

建议实现两种：

1. 固定温度扫场：

   * 从负场到正场
   * 再从正场扫回负场

2. 固定场降温：

   * 检查弱场与强场下纹理是否截然不同

### 11.9 观测量实现

#### 平均极化

```python
m = np.mean(s)
```

#### 结构因子

```python
fluc = s - np.mean(s)
Sq = np.abs(np.fft.fftshift(np.fft.fft2(fluc)))**2
```

#### 径向平均功率谱

将 `Sq` 按 (|q|) 分箱平均，得到 `S(q)`。

#### 畴尺度

从径向平均谱峰值位置 `q_peak` 估算：

```python
L_dom = 2*np.pi / q_peak
```

### 11.10 toy 扫描建议

由于当前单层应视为三角晶格，toy 扫描不应再使用方格子上的 checkerboard 作为主参考态，而应改为比较：

* Gamma 均匀态；
* M 点条纹态；
* K 点三子晶格态。

推荐先做以下几组：

#### CIPS-like

* 深局域势
* 较大 J1
* 小 J2
* 中等 D_eff
* 预期更偏向 Gamma 均匀 FE 或大畴态

#### CIPSe-like

* 浅局域势
* 中等 J1
* 较大 J2
* 中等或略小 D_eff
* 预期更容易在 M 或 K 通道上出现有限尺度纹理

第一阶段参数扫描建议按以下顺序推进：

1. 固定 D_eff = 0，只扫 J2/J1，先看纯短程竞争是否已能把体系从 Gamma 推向 M 或 K；
2. 再逐步打开 D_eff，观察偶极项是否把离散条纹态扩展成更真实的有限尺度纹理或迷宫态；
3. 最后再加外场 h，检查强场是否能把 M 或 K 纹理压回近乎均匀态。

对程序实现而言，推荐至少提供以下三类初始化：

* init_gamma()：全同号；
* init_stripe_M(direction)：三角晶格三个等价 M 方向之一的条纹态；
* init_K_three_sublattice(mode='A,A,-A')：三子晶格 K 态。

这样既可用于低温能量比较，也可作为 Monte Carlo 或动力学初态检查多稳态。

### 11.13 第一阶段验收标准

代码完成后，至少应能展示：

1. CIPS-like 参数更偏均匀 FE / 大畴；
2. CIPSe-like 参数更容易出现有限尺度纹理；
3. 随降温，非均匀结构因子增强；

只要这六点中前四点已经能稳定出现，代码与模型路线就已经达标。

## 13. 最终结论

对当前问题，一套合理的格点模型是：

[
H=
\sum_i \left(\frac{\alpha}{2}s_i^2-\frac{\beta}{4}s_i^4+\frac{\gamma}{6}s_i^6\right)
-J_1\sum_{\langle ij\rangle}s_is_j
+J_2\sum_{\langle\langle ij\rangle\rangle}s_is_j
+\frac12\sum_{i\neq j}D_{\rm eff}\frac{s_is_j}{r_{ij}^3}
-h\sum_i s_i.
]

它的主要优点是：

* 参数物理意义清楚；
* 与 DFT 参数提取自然衔接；
* 能快速做 toy 测试；
* 可以直接输出畴纹理与功率谱；
* 便于 code agent 立即实现。

Toy 阶段最关键的不是参数是否精确，而是能否先验证：

* CIPS 更偏向均匀 FE / 大畴；
* CIPSe 更偏向竞争纹理 / 有限尺度结构；
* 弱场冷却下复杂纹理增强，强场下被压制。

只要这一层成立，后续用 DFT 参数替换 toy 参数就是一条明确而可执行的路线。
